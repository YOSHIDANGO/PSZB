# DIE YET SLOT v12 Layout Fix

v12では、最新スクショの指摘に合わせて大きく座標を直しています。

## 修正内容

- 液晶
  - 左右の幅をかなり狭く調整
  - 上下は少し広げて、下の余白を減らす
- リール
  - 左右の幅を大きく広げる
  - 実機リール窓に近い位置へ移動
- LEVER ON
  - 小さいSTARTボタンではなく、左下の黒いレバー/ノブ部分へ移動
- PUSH
  - 中央のPUSHボタンのまま
- MAX BET
  - 赤いMAX BETボタン
- STOP
  - 赤3ボタンに対応

## 起動

```bash
python -m http.server 8080
```

通常:
`http://localhost:8080`

位置確認:
`http://localhost:8080?debug=1`

debug時:
- 水色: 液晶
- 黄色: リール
- 緑: LEVER
- オレンジ: PUSH
- 赤: MAX BET / STOP
