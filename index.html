    この機種について

    ゾンビだらけの街を舞台にした5ラインのAタイプです。通常時は青7を目安に狙い、滑りや停止形からCHERRY・AMMO・HERO・ボーナスを察知します。

    基本の流れ

    1. MAX BETで3枚投入し、レバーでリールを始動します。
    2. REPLAYとBELLは基本役、CHERRY・AMMO・HEROはチャンス役です。
    3. レア役の一部からボスとのBONUS CHANCEへ発展します。
    4. BONUS確定後は、赤7・青7・BARの対応図柄を揃えるとボーナス開始です。

    リーチ目

    通常時には出にくい特殊な停止形です。出現した時点でボーナス確定となり、液晶に「リーチ目 / BONUS確定」が表示されます。

    例：RED7・REPLAY・BAR / BLUE7・CHERRY・RED7 / HERO・BAR・HERO

    SURVIVE MODE

    SBB終了後は64G、BIG終了後は32G継続する高期待モードです。滞在中はボーナス抽選が強化され、専用BGMと筐体ランプで示唆します。REG終了後は突入しません。
